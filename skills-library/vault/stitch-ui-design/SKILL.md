# Stitch UI Design

## Purpose
Generate highly effective, detailed prompts to create professional user interfaces using **Google Stitch**. Bridge the gap between vague ideas and specific, high-fidelity design outputs.

## Instructions

### 1. Prompt Structuring
Structure prompts for Stitch following this pattern:
- **Role/Context**: "Act as a senior UI designer..."
- **Task**: "Create a dashboard for..."
- **Style/Aesthetic**: "Clean, minimalist, using a pastel color palette..."
- **Components**: List specific elements (e.g., "Sidebar navigation, Data table with filters, User profile card").

### 2. Specificity Strategy
- **Avoid**: "Make it look good."
- **Use**: "Use a 4-column grid, 8px rounded corners, and Inter font."
- **Layout**: Define the layout clearly (e.g., "Two-column bento grid").

### 3. Iteration
- Analyze Stitch outputs.
- Suggest refinements based on design principles (Alignment, Contrast, Hierarchy).

### 4. Cases
- **Mobile**: Specify "Mobile-first", "Touch targets > 44px".
- **Dashboard**: Focus on "Data visualization", "Widgets", "Clear metrics".
- **Landing Page**: Focus on "Hero section", "Call to Action (CTA)", "Social proof".

## Constraints
- This skill does NOT generate code directly; it generates **prompts** for Stitch.
- Always ask for the target device (Desktop/Mobile).
