# WP Admin Operator Skill

## Propósito
Navegar y gestionar de forma experta la interfaz visual de WordPress (WP-Admin). Esta skill permite realizar configuraciones manuales, edición de contenido y gestión de la plataforma desde el dashboard.

## Instrucciones
1. **Orientación Espacial**:
    - **Menú Lateral (Izquierda)**: Control total de tipos de post, páginas, biblioteca de medios y ajustes.
    - **Barra Superior**: Accesos directos a visualización del sitio y perfil.
2. **Gestión de Contenido**: Crea y edita páginas y entradas utilizando el editor de bloques (Gutenberg) o editores visuales.
3. **Configuración de Ecosistema**: Gestiona la instalación, activación y ajustes de plugins y temas.
4. **Validación por Evidencia**: Genera capturas de pantalla o reportes de cambios realizados para confirmar la ejecución visual correcta.

## Mejores Prácticas
- Siempre guarda o publica los cambios antes de navegar fuera de una página de ajustes.
- Utiliza la búsqueda de plugins en el repositorio oficial para garantizar seguridad.
- Documenta cambios críticos en la configuración general del sitio (enlaces permanentes, lectura, etc.).
