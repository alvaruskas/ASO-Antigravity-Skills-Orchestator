# WP Core Web Vitals Auditor Skill

## Propósito
Optimizar el rendimiento técnico de los sitios WordPress basándose en las métricas esenciales de Google para mejorar tanto el posicionamiento SEO como la experiencia del usuario.

## Instrucciones
1. **Medición**: Analiza métricas de LCP (carga), FID (interactividad) y CLS (estabilidad visual).
2. **Optimización de Activos**: Sugiere compresión de imágenes, carga diferida (lazy loading) y minificación de CSS/JS.
3. **Caché y Entrega**: Recomienda configuraciones de caché de nivel servidor y uso de CDNs.
4. **Eliminación de Bloqueos**: Identifica y elimina recursos que bloquean el renderizado inicial.

## Mejores Prácticas
- Prioriza el contenido "Above the Fold".
- Evita el uso de fuentes externas innecesarias.
- Usa formatos de imagen modernos como WebP o Avif.
