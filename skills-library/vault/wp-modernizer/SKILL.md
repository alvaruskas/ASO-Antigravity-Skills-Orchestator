# WP Modernizer Skill

## Propósito
Llevar proyectos antiguos de WordPress a la era moderna. Se enfoca en la refactorización de código legado, mejora de compatibilidad con versiones recientes de PHP y aplicación de estándares modernos de codificación de WordPress.

## Instrucciones
1. **Auditoría de Versiones**: Identifica funciones obsoletas (deprecated) y versiones de PHP requeridas.
2. **Refactorización Segura**: Actualiza bucles, hooks y llamadas a la base de datos sin romper la funcionalidad.
3. **Escalabilidad**: Sugiere el uso de APIs modernas (REST API, Gutenberg Blocks) en lugar de implementaciones manuales antiguas.

## Mejores Prácticas
- Siempre prueba la compatibilidad con PHP 8.1+.
- Documenta cada cambio de función obsoleta para referencia futura.
- Mantén la retrocompatibilidad cuando el entorno lo exija.
